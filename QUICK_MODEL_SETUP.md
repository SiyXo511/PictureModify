# 快速设置模型文件

## 当前状态

模型文件还没有下载。按照以下步骤操作：

## 步骤1：让程序下载模型

### 方法A：运行程序并触发下载（推荐）

1. 运行程序：
   ```bash
   python src/main.py
   ```

2. 打开一张包含文字的图片

3. 选择包含文字的区域

4. 点击"文字识别"按钮

5. 程序会自动下载模型（首次使用，需要几分钟）

6. 等待下载完成

### 方法B：使用Python命令直接下载

在命令行运行：

```bash
python -c "from paddleocr import PaddleOCR; ocr = PaddleOCR(use_angle_cls=True, lang='ch')"
```

这会触发模型下载。

## 步骤2：复制模型到项目目录

模型下载完成后，运行：

```bash
python download_models.py
```

脚本会自动：
- 查找已下载的模型文件
- 复制到 `models/paddleocr/` 目录

## 步骤3：验证

检查 `models/paddleocr/` 目录是否包含：
- `det/` - 检测模型
- `rec/` - 识别模型
- `cls/` - 方向分类模型（可选）

## 完成！

现在模型文件已经在项目目录中了，可以：
- 提交到Git仓库（如果使用）
- 分享给团队成员
- 其他人使用时无需下载

## 注意事项

- 模型文件大小约100-200MB
- 首次下载需要网络连接
- 下载可能需要几分钟时间
